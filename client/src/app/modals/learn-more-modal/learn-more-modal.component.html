This is the pet project based on the udemy course

The main goal of this - is the practical skillup in both Angular 18 and .net 8.0

  • Frontend - Angular 18 (bootstrap, font awesome, signalR)
  • Backend - C# .net core 8.0 (entity framework, identity, mvc, n-tier architecture, signalR)
  • Integration with 3rd party services (cloudinary)
  • Hosting - Azure (web app, sql database, CI)
  • Postman - included collection with API calls for all of the endpoints
  • Project source code on GitHub
